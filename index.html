Welcome to the Sage SData 2.0 on GitHub

This repository provides primarily technical information on the upcoming version 2.0 of the standard.

SData is successfully used to deliver a number of key projects and is recognized as an important building block in the Sage technology strategy. The version 2.0 of the standard focuses its attention on needs beyond the traditional application integration - it provides new aspects specifically aimed towards the needs of web product and interface definitions. The main aspects of the new release are:

· Maintaining compatibility with the SData 1.1 version. This means that SData 1.1 implementations are automatically version 2.0 compliant · Specification of the JSON format for SData information exchanges · Decreasing number the restrictive aspects imposed by SData, resulting in a smaller and better defined core of the specification · Stressing the freedom of choice for development teams and architects · Clarifying the role and form of contracts

 

The documents provided, highlight a number of import aspects of the standard as a preview on the upcoming release. We will continue to update and extend the information as it becomes available.
